Bootstrap Slider Knockout Binding

The Knockout binding allows you to use an observable as a value for the slider and to customize the options of the slider via knockout params.

You can find the Bootstrap Slider project here: https://github.com/seiyria/bootstrap-slider

Examples

Checkout the Examples page for a simple example of how a page using the bindings should look like.
